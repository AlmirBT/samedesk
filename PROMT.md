# FunTime HelpDesk — Frontend Development Prompt

## 🎯 Задача
Разработать современный, анимированный **frontend** для системы автоматизации технической поддержки **FunTime HelpDesk** на **React**. Только фронтенд — без бэкенда, все данные mock/static.

### Типографика
- Заголовки: `Syne` (Google Fonts) — геометричный, современный
- Тело: `DM Sans` — читаемый, нейтральный
- Моно: `JetBrains Mono` — для кодов/ID

### Принципы дизайна
- Тёмная тема как основная
- Красные акценты на активных элементах, hover-эффектах, иконках
- Glassmorphism для карточек: `backdrop-filter: blur(12px)`, полупрозрачные фоны
- Анимации: плавные, 200-350ms, `cubic-bezier(0.4, 0, 0.2, 1)`
- Sidebar со свечением при hover
- Красные градиенты на кнопках CTA
- Тонкие красные border-left у активных пунктов меню

## ⚡ Анимации (обязательно)

### Глобальные
- `framer-motion` или CSS transitions для всех переходов между страницами
- Staggered появление карточек/строк таблицы (задержка 50ms между элементами)
- Пульсирующий красный dot у новых обращений
- Skeleton loading для всех загружаемых данных
- Hover: `translateY(-2px)` + усиление тени для карточек
- Smooth scroll везде

### Специфичные
- Sidebar: иконки с небольшим `scale(1.1)` при hover + tooltip
- Бейджи с числами: анимация при изменении значения (`scale` 1→1.2→1)
- Статус-индикаторы сотрудников: мигающий зелёный/красный dot
- Прогресс-бары смен: анимированное заполнение при появлении
- Приоритет обращения: цветные полосы слева от карточки

## 🗂 Структура страниц

### 1. Страница авторизации (`/login`)
- Центрированная форма на тёмном фоне
- Логотип "FunTime HelpDesk" сверху с красным акцентом
- Поля: Логин, Пароль
- Кнопка "Войти" — красный градиент
- Второй шаг: 2FA код (6 цифр, auto-focus на каждую)
- Анимация: форма появляется снизу с fade

### 2. Главная страница (`/dashboard`)
Виджеты (grid 3 колонки):
- Открытые обращения (число + trend)
- Задачи на сегодня
- Активные сотрудники на смене
- Среднее время ответа
- График обращений за 7 дней (recharts LineChart)
- Последние 5 обращений (мини-список)

### 3. Обращения (`/tickets`)
Левая панель (список):
- Фильтры: Все / Мои / По тегу / По важности
- Поиск
- Карточка обращения: аватар игрока, ник, платформа (ВК/ТГ/ДС иконка), первая строка сообщения, время, приоритет (цвет), теги
- Цветовая полоска слева по приоритету: серый/зелёный/жёлтый/оранжевый/красный

Правая панель (чат):
- Шапка: ник игрока, платформа, теги, кнопки действий
- История сообщений (пузыри: игрок слева, сотрудник справа)
- Поддержка вложений (превью картинок)
- Внутренний чат заявки (переключатель)
- Нижняя панель: поле ввода + кнопки: Шаблоны, Прикрепить, Отправить
- Правая панель инфо: важность, теги, данные игрока, статистика с сервера

### 4. Задачи (`/tasks`)
- Kanban-доска ИЛИ таблица (переключатель)
- Колонки: Новые / В работе / На проверке / Закрыты
- Карточка задачи: заголовок, дедлайн, важность, ответственный, теги
- Drag & drop между колонками (опционально)

### 5. Пользователи (`/users`)
- Таблица: аватар, ник, имя, роли, статус (онлайн/офлайн/на смене), кол-во обращений
- Фильтр по ролям
- Клик → drawer/модал с профилем пользователя
- Кнопка добавить пользователя

### 6. Роли (`/roles`)
- Карточки ролей с иконками
- Список прав чекбоксами (читать/отвечать/блокировать и тд)
- Создать новую роль (модал)

### 7. Теги (`/tags`)
- Список тегов с цветом и счётчиком использования
- Статусы тега: Создан / Обработан / Кастомный
- Создание/редактирование тега

### 8. Смены (`/shifts`)
- Расписание: timeline/calendar вид
- Карточка смены: время, сотрудники, оплата в токенах
- Кнопка "Готов" (активна за 10 мин до смены)
- Текущая смена: таймер, кнопки "Перерыв" / "Завершить"
- Логи активности

### 9. Настройки (`/settings`)
- Вкладки: Профиль / Уведомления / Автоответы / Шаблоны / Система
- Настройка автоответов: ключевые слова + ответ + исключения
- Шаблоны быстрых ответов

### 10. Статистика (`/stats`)
- Фильтр периода
- Таблица сотрудников с метриками
- Цветовая подсветка: зелёный (>25% выше среднего), красный (<25% ниже)
- Графики: recharts BarChart, LineChart, PieChart
